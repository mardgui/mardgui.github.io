# Quentin Guimard - Academic Portfolio

🌐 **[mardgui.github.io](https://mardgui.github.io)**

## ✨ Features
* **Perfect Performance:** 100/100 Lighthouse scores across the board. Zero layout shifts (CLS), fully self-hosted variable fonts, and zero render-blocking CSS.
* **Accessible:** Passes strict WCAG contrast ratios and full screen-reader compliance.
* **Component-Driven:** Easily modularized sections for Publications, Projects, and Timeline.
* **Light/Dark Mode:** Native Tailwind integration responsive to system preferences.

## 🚀 Local Development

1. Clone the repository and install dependencies:
```bash
pnpm install
```
2. Start the local development server:
```
pnpm dev
```
3. Build for production:
```
pnpm build
```
